#  Student Management System

A full-stack **Student Management System** built using **Flask, SQLite, and Bootstrap**.
This web application allows administrators to manage student records through a secure login system and a modern dashboard interface.

---

##  Features

*  Admin login authentication
*  Add new student records
*  View all students in a structured table
*  Edit student details
*  Delete student records
*  Dashboard showing total number of students
*  Modern responsive UI using Bootstrap
*  SQLite database for easy deployment

---

## Tech Stack

**Backend**

* Python
* Flask

**Frontend**

* HTML
* CSS
* Bootstrap

**Database**

* SQLite

**Deployment**

* Render (recommended)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/vamsiveduruparthi/student_management.git
cd student_management
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application

```bash
python app.py
```

Open your browser and go to:

```
http://127.0.0.1:5000
```
